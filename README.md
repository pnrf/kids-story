### Как установить и запустить проект:

* Клонировать репозиторий:

```console
    git clone https://github.com/pnrf/mesto.git
```

* Установить зависимости:

```console
    npm install
```

* Собрать проект Вебпаком:

```console
    npm run build
```

* Запустить проект на локальном сервере:

```console
    npm run dev
```
